#Benzene Framework

## What is Benzene Framework

Benzene framework is micro web framework written in java.

## Features

* Simple Routing
* Start Server with very little code

## Requirements

* Java 1.7+

## file structure

* BenzeneFramework - Framework module
* Jade4Benzene     - View engine module (jade)
* src/test         - test code
